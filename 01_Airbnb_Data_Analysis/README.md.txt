<h1> 🏡 Airbnb Data Analysis </h1>

<h2> 📌 Project Overview </h2>
<p> This project focuses on cleaning and analyzing Airbnb listings data to extract insights on pricing trends, seasonal patterns, and customer reviews.</p>

<h2> 🛠️ Skills & Tools Used </h2>
<p> - Python: Pandas, NumPy, Matplotlib, Seaborn </p>
<p> - SQL: Data querying and filtering </p>
<p> - Data Cleaning: Handling missing values, outliers, and formatting </p>
<p> - Exploratory Data Analysis (EDA): Identifying patterns and trends </p>

<h2> 📊 Dataset </h2>
<p> - <b>Source</b>: <a href="https://www.kaggle.com/datasets/airbnb/seattle">Kaggle - Airbnb Dataset</a> </p>
<p> - <b>Description</b>: Contains Airbnb listing details, pricing, availability, and customer reviews. </p>
